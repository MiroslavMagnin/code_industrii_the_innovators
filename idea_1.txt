Для получения информации о дорогах, условиях и их особенностях и тд будет использоваться API mapbox. Мы будем использовать для определения миниимального пути алгоритм A-star, разработанный специально для нахождения пути между двумя точками. Для того, чтобы получить ответ на задачу, исходя из полученных данных с помощью алгоритмов, будет использоваться динамическое программирование. Ответом на задачу будет оптимальная суммарная стоимость пути, полученная с помощью динамического программирования. Код эффективен постольку, поскольку используется алгоритм A-star и динамическое программирование
